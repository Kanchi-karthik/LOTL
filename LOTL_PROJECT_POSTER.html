LOTLGuard: Living-Off-The-Land Threat Detection Framework

ML-Driven Real-Time SOC Dashboard with MITRE Mapping, Quarantine, and Incident Response

Objective

  • Detect LOTL command misuse in real time using behavioral + anomaly signals.
  • Correlate Falco-style events with threat scoring and MITRE ATT&CK technique mapping.
  • Provide a SOC dashboard for live incidents, topology views, and attacker geolocation.
  • Trigger automated incident response with quarantine and forensic log capture.
  • Support local simulation and scalable cloud-native deployment paths.

Core Modules

  • Dashboard layer: Flask UI, APIs, live charts, and incident timeline.
  • Backend pipeline: parsing, enrichment, threat scoring, and persistence.
  • Detection engine: Isolation Forest + command feature extraction.
  • Response engine: severity-based containment and auto-block logic.
  • Simulation lab: shared kernel command injection and attack playback.

Implementation Highlights

  • Language and runtime: Python 3.8+, Flask, scikit-learn, joblib.
  • Features: command entropy, token patterns, dangerous-tool usage, network indicators.
  • Storage: live event logs + persistent quarantine database with forensic snapshots.
  • Monitoring: Falco-compatible stream ingestion and Kubernetes-aware collectors.
  • Visualization: topology graphs, command analytics, and geolocation map layers.

Results

  • Model accuracy: approximately 95% on test data.
  • False positive rate tuned to around 2% for balanced alerting.
  • Real-time detection latency: under 50 ms per prediction.
  • Threat score output normalized to 0-100 for clear SOC triage.
  • Automated response routes: low, medium, high, critical severities.
  • Persistent quarantine evidence via per-incident forensic JSON logs.

Conclusion

LOTLGuard demonstrates a practical and reproducible approach for identifying living-off-the-land attacks by combining machine learning, rule-based scoring, and response orchestration. The framework links simulated or live kernel telemetry to actionable SOC intelligence through visualization, containment, and MITRE-aligned reporting.
